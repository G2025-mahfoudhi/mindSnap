# Hero Video Keyframes — Variation A "Connaissance qui s'organise"

## Spécifications communes
- Format : 1920×1080 (16:9)
- Style : Flat illustration, vector art type Undraw
- Palette : Teal #3d7f7e, Sage #7FA89B, Lavender #C7B8EA, Pink #F4C2C2, Cream #F7F5F4
- Aucun texte, aucun visage, pas d'UI

---

```
Flat illustration, 1920x1080, 16:9 aspect ratio, clean vector art style like Undraw. Soft pastel palette dominated by teal #3d7f7e and sage green #7FA89B, with accents of lavender #C7B8EA and soft pink #F4C2C2. Cream #F7F5F4 solid background, very clean. High quality, 4K, no watermark.
```

## Prompt vidéo — Veo 3.1 Flash

```
A serene flat-vector illustration animation, 1920x1080, 10 seconds, smooth slow-motion. Cream #F7F5F4 background throughout. Opening shot: dozens of white document cards, soft-yellow sticky notes, and teal folder icons float randomly across the frame, each slightly rotated, with small teal and lavender light particles drifting between them. Over 5 seconds, all elements begin a gentle inward motion — documents slowly drift and rotate toward the center of the frame, particles stream inward along soft curved arcs, motion is fluid and weightless like objects in zero gravity. The background remains static cream. At 5-7 seconds, elements converge into a neat organized stack at center while a soft radial glow in teal and lavender emerges behind the stack, growing in intensity. Final 3 seconds: everything is settled and calm, the stack glows gently, a few stray particles orbit lazily. No text, no faces, no camera movement — only the floating objects move. The overall mood is peaceful, meditative, "scattered thoughts becoming organized knowledge". Pastel palette only: teal #3d7f7e, sage #7FA89B, lavender #C7B8EA, soft pink #F4C2C2, cream #F7F5F4. Flat 2D vector style, no 3D depth, no realistic textures. Seamless loop potential — the final frame can reverse back to dispersion.
```

### Négatif prompt (si supporté par l'outil)
```
No text, no letters, no words, no watermarks, no logos, no faces, no people, no realistic 3D, no shadows on background, no camera pan, no zoom, no glitch, no flickering, no dark colors, no black background.
```

## Usage vidéo

1. Générer Frame 1 et Frame 2 avec un générateur d'images (Flux, DALL-E, Midjourney...)
2. Option A — Veo 3.1 Flash text-to-video : utiliser le prompt ci-dessus directement
3. Option B — Veo 3.1 Flash image-to-video : fournir Frame 1 comme image de départ + prompt simplifié :
   ```
   The document cards, sticky notes and folder icons slowly float and drift toward the center of the frame, converging into a neat organized stack over 10 seconds. A soft radial glow in teal and lavender fades in behind the stack. Gentle slow-motion, weightless, peaceful. No camera movement. Cream background stays fixed.
   ```
4. Durée cible : 10 secondes, seamless loop
5. Export : `.mp4` H.264 + `.webm` VP9, 1920×1080, 5-8 MB
